# Dystopia
Dystopia is an experimental discord bot that is currently being developed.

## Current Commands
`[] - required argument`<br />
`() - optional argument`<br />

### Admin
`/unban [ID]`<br />

### Utility
`/serverinfo`<br />
`/about`<br />

### Messages
`/clear [number]`<br />

### Roles
`/addrol [user] [roles]`<br />
`/delrole [user] [roles]`<br />
/mute [user] (reason)<br /> /deafen [user] (reason)<br />
